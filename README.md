# Dashboard Amazônia 2026

Dashboard público da Expedição Amazônia 2026.

## Publicação no GitHub Pages

1. Envie os arquivos deste pacote para a raiz do repositório.
2. No GitHub, acesse `Settings > Pages`.
3. Em `Build and deployment`, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. Salve e aguarde o GitHub gerar a URL pública.

O arquivo principal do site é `index.html`.
